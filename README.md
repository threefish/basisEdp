BasisEdp 基于[Basis MVC](https://github.com/threefish/basisMvc "Basis MVC")的企业开发框架
================================================================================
项目目前处于开发中（_空闲时间码代码_）
======
**已通过压力测试**

**功能列表**
*   集成 Apache Shiro 安全权限框架
*   后台管理界面采用 Bootstrap + adminLTE
*   AOP注解式事务 、日志
*   AOP支持类切面、方法切片（一切由你控制）
*   采用 Beetl 视图模板引擎
*   集成 Quartz 后台定时任务
*   集成 Lucene 全文检索搜索引擎
*   集成 Email 服务
